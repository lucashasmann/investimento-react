# Investimento

Simulador de rendimento de investimentos com juros compostos.

## 🔧 Funcionalidades
- Cálculo de rendimento com base em:
  - Aporte inicial
  - Aportes mensais
  - Taxa de juros
  - Tempo em meses
- Exibição do valor final acumulado

## 🛠️ Tecnologias Usadas
- React Native
- React
- Expo
- JavaScript

## ▶️ Como Executar
```bash
git clone https://github.com/seu-usuario/investimento
cd investimento
npm install
npm start
```
